Install Node JS Software

Go to terminal or command line.

Execute following commands to run this application.

```
$ npm install -g cordova ionic

$ git clone https://github.com/srinivastamada/ionic2-camera.git

$ cd ionic2-camera

$ npm install

$ ionic serve

```
